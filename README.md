# 🎬 Netflix Clone

A responsive Netflix-inspired web application built using **HTML**, **CSS**, and **JavaScript**. This project recreates the modern Netflix user interface with an engaging design, interactive movie cards, and a seamless browsing experience.

## 🚀 Live Demo

🌐 https://owais-core03.github.io/netflix-clone/

## 📂 GitHub Repository

🔗 https://github.com/owais-core03/netflix-clone

## ✨ Features

* 🎥 Netflix-inspired homepage
* 📱 Fully responsive design
* 🎬 Movie browsing interface
* 🔍 Search functionality
* ❤️ Clean and modern UI
* ⚡ Smooth animations and hover effects
* 💻 Cross-browser compatibility

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/owais-core03/netflix-clone.git
```

2. Navigate to the project folder

```bash
cd netflix-clone
```

3. Open `index.html` in your preferred browser.

Or simply use **Live Server** in Visual Studio Code.
